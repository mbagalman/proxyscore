# Proxyscore Examples

This folder contains runnable examples demonstrating how to use `proxyscore`.

Versioned TOML examples for the `proxyscore` CLI are in [`configs`](configs/audit.toml):

- `audit.toml`
- `compare.toml`
- `baseline.toml`
- `monitor.toml`

- `01_customer_health.py`: A basic, end-to-end audit of a customer health score using the included synthetic dataset.
