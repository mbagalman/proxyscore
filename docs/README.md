# Documentation

Project documentation for `proxyscore`.

## Layout

- **`internal/`** — working notes that are *not* published with the package
  (kept out of version control via `.gitignore`): the original project brief,
  background research, and code-review records.

User-facing documentation currently lives in the top-level
[`README.md`](../README.md) and [`CHANGELOG.md`](../CHANGELOG.md). Additional
public docs (guides, API reference) will be added to this folder as the project
grows.
